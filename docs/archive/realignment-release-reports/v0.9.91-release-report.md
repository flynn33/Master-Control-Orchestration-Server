# Release Completion Report — v0.9.91

Browser dashboard mirror of v0.9.90's relative-time heartbeat formatting.

## Scope completed

### `formatRelativeUtcTime` JavaScript helper + dashboard Supervisor card rewrite

v0.9.90 added the relative-time helper on the WinUI Shell. v0.9.91 ships the equivalent on the browser dashboard so both surfaces present the same operator-friendly heartbeat string.

JavaScript helper in `resources/web/app.js`:

```js
function formatRelativeUtcTime(isoStampUtc) {
  if (!isoStampUtc || typeof isoStampUtc !== 'string') return '';
  const parsed = Date.parse(isoStampUtc);
  if (Number.isNaN(parsed)) return '';
  const deltaMs = Date.now() - parsed;
  if (deltaMs < 0) return 'just now';
  const sec = Math.floor(deltaMs / 1000);
  if (sec < 60) return sec + 's ago';
  if (sec < 3600) { ... 'Xm Ys ago' / 'Xm ago' ... }
  if (sec < 86400) { ... 'Xh Ym ago' / 'Xh ago' ... }
  ... 'Xd Yh ago' / 'Xd ago' ...
}
```

Bucket boundaries and output strings match the WinUI Shell helper exactly so the two surfaces show identical text for the same heartbeat.

`renderSupervisorPanel` connected branch updated:

```js
// before
statusLine = 'Mode: ' + mode + ' · Status: connected'
  + (s.lastHeartbeatUtc ? ' · Last heartbeat (UTC): ' + s.lastHeartbeatUtc : '');

// after
let heartbeatPhrase = '';
if (!s.lastHeartbeatUtc) {
  heartbeatPhrase = ' · Last heartbeat: (none yet)';
} else {
  const relative = formatRelativeUtcTime(s.lastHeartbeatUtc);
  heartbeatPhrase = relative
    ? (' · Last heartbeat: ' + relative)
    : (' · Last heartbeat (UTC): ' + s.lastHeartbeatUtc);
}
statusLine = 'Mode: ' + mode + ' · Status: connected' + heartbeatPhrase;
```

Empty heartbeat shows `(none yet)`. Parse failure falls back to the raw UTC stamp (consistent with the shell behavior). Time passes between dashboard polls so the displayed phrase ages naturally on the 8s refresh cadence.
