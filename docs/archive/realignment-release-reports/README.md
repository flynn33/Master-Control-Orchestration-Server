# Archived realignment release and phase reports

**Historical evidence — not current instructions.** This directory holds:

- **Per-release reports** for the v0.9.x and v0.10.x lines (`v0.9.*-release-report.md`, `v0.10.*-release-report.md`), moved from `handoff/realignment/`.
- **Phase completion reports** for PHASE-00 through PHASE-11 (`PHASE-*-completion-report.md`), the delivered realignment program phases.

Each report describes the repository as it was at that release or phase boundary. File paths, port numbers, and feature states quoted inside them may since have changed — check the [current wiki](../../wiki/) and [`VERSION.json`](../../../VERSION.json) before acting on anything here.

Active phase plans, the program manifest, and reports for the current v0.11.0 line remain in [`handoff/realignment/`](../../../handoff/realignment/).
