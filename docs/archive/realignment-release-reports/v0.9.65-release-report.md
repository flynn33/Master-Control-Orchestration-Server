# Release Completion Report — v0.9.65

Three more in-tree backends: persistent-context (JSON-backed K/V), knowledge-graph (JSON-backed entities + relations), file-watcher (polling snapshot diff). Reachable count goes 17/28 → 20/28; gateway tools grow from 44 to 57.

## Scope completed

* `Specialization::PersistentContext` / `KnowledgeGraph` / `FileWatcher` enum values + CLI parser branches.
* `mcosDataDirectory()` resolves `%ProgramData%\MCOS` via `GetEnvironmentVariableW` and `create_directories` on demand.
* **persistent-context**: `ctx.set` / `ctx.get` / `ctx.list` / `ctx.delete` over JSON file at `%ProgramData%\MCOS\persistent-context.json`. Process-wide mutex around the read-modify-write cycle.
* **knowledge-graph**: `kg.entity.upsert` (observation merging by exact-string dedup) / `kg.relation.create` (idempotent on (from,to,relationType)) / `kg.search` (case-insensitive substring across name + entityType + observations, surfaces related relations) / `kg.read_graph` / `kg.delete_entity`.
* **file-watcher**: `watch.add` / `watch.poll` / `watch.list` / `watch.remove`. `takeSnapshot()` does iterative-stack `FindFirstFileW` walk capturing `{sizeBytes, lastWriteTime}` per file; `watch.poll` re-walks and computes added/modified/removed against the previous snapshot, then atomically replaces.
* `MasterControlRuntime` upserts 3 supervised pools.
