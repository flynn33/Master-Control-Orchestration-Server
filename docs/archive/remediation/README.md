# Archived remediation program

**Historical evidence — not current instructions.** These files document the pre-realignment remediation program (the "gut and rebuild" era) and its workstream completion reports. The architecture they describe predates the gateway-first realignment locked by ADR-002/ADR-003; do not implement from these documents.
