# Release Completion Report — v0.9.96

Tiny operator-visibility add on the WinUI Supervisor card detail row.

## Scope completed

The v0.9.79 supervisor detail row carried Assignment / Config / Expires / Last heartbeat / Client lines. v0.9.96 inserts an `Issued: 3m 14s ago` line between Config and Expires so the operator can see assignment age at a glance — useful when deciding whether to rotate a long-running config.

```
Assignment: SUP-20260511T011742Z-63b8
Config: CFG-20260511T011742Z-6ba4
Issued: 9m 14s ago             ← NEW v0.9.96
Expires: in 2h 14m (2026-05-...)
Last heartbeat: 5s ago
Client: claude-supervisor-client
```

Uses the v0.9.90 `formatRelativeUtcTime` helper. Falls back to raw UTC stamp on parse failure.
