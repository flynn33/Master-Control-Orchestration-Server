v0.9.94 shipped `formatFutureUtcTime` on the WinUI Shell so the Overview Supervisor card detail row renders `expiresAtUtc` as `"Expires: in 2h 14m (2026-05-10T...)"`. v0.9.95 ships the JavaScript equivalent so the browser dashboard's Supervisor card achieves the same pattern.

function formatFutureUtcTime(isoStampUtc) {
  if (!isoStampUtc || typeof isoStampUtc !== 'string') return '';
  const parsed = Date.parse(isoStampUtc);
  if (Number.isNaN(parsed)) return '';
  const deltaMs = parsed - Date.now();
  if (deltaMs <= 0) {
    const since = formatRelativeUtcTime(isoStampUtc);
